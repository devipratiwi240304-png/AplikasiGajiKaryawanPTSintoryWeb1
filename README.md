# Aplikasi Gaji Karyawan PT Sintory

Aplikasi berbasis web untuk mengelola data karyawan dan penggajian, 
meliputi input data karyawan, perhitungan gaji, serta laporan gaji.

## Fitur Aplikasi
- Manajemen Data Karyawan
- Input dan Simpan Data Gaji
- Laporan Gaji Karyawan
- Tampilan antarmuka sederhana dan mudah digunakan
